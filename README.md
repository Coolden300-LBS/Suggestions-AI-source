# VIKTIGT! LÄS:
Projäkt kräver python 3.12 för att fungera och flera andra libraries. Du kan se de i 'main.py' filen.
Var snäll och skapa en virtual enviroment med python 3.12, lägg den här mappen inne i enviroment och 
installera de libraries med hjälp av pip för din enviroment eller vad som helst du använder.
