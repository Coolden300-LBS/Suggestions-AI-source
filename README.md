# VIKTIGT! LÄS INNAN DU ANVÄNDER:
Projekt kräver python 3.12 för att fungera och flera andra biblioteker. Du kan hitta de biblioteker som krävs i 'main.py' filen.
Var snäll och skapa en virtual environment med python 3.12, lägg den här mappen inne i enviroment och installera de krävande biblioteker med hjälp av pip eller vad som helst du använder.
Sedan kör bara 'main.py' filen.
Tyvärr filen går inte att starta utan en editor så som Visual Studio Code eller annat.
